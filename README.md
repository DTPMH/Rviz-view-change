# Rviz-view-change
这个仓库是可以自动改变rviz视角的例子   
**ubuntu18.04教程**   
**安装**   
**命令行安装**   
1. sudo apt install ros-<ros-distro>-rviz-animated-view-controller   
2. cd catkin_ws/src   
3. git clone git@github.com:DTPMH/Rviz-view-change.git   
4. catkin_make
5. rosrun change-view change-view
